Описание проекта

Мы создали Telegram Web App для фермерского магазина "ЭкоФерма". Цель — предоставить удобный интерфейс для покупки продуктов.

  • Функции: Интерактивный каталог, корзина, оплата через Telegram Payments, отслеживание заказов.
  • Интеграции: ЮKassa для платежей, уведомления через Telegram.
  • Результат: Средний чек вырос на 20%, время оформления заказа сократилось на 50%.
